# 고양아

아래 예제와 같이 강아지를 출력하시오.
지

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

```java
public class Main {

    public static void main(String[] args) {
        System.out.println("|\\_/|\n");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}
```
